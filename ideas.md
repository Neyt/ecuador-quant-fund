## Idea 1: "Terminal Financiero" — Estética Data-Dense Institucional

**Movimiento de Diseño:** Inspirado en terminales Bloomberg/Reuters y dashboards de trading institucional, con una estética de "control room" financiero.

**Principios Centrales:**
1. Densidad informativa controlada — mucha información, pero jerárquicamente organizada
2. Tipografía monoespaciada para datos, serif para narrativa
3. Fondo oscuro navy profundo (#0a0e27) con acentos de verde terminal (#00ff88) y ámbar (#ffd700)
4. Micro-animaciones de "datos en vivo" — números que parpadean, líneas que se dibujan

**Filosofía de Color:** Navy profundo como base (confianza institucional), verde neón para datos positivos (crecimiento), rojo suave para drawdowns, ámbar/dorado para títulos y acentos premium.

**Paradigma de Layout:** Layout asimétrico con sidebar de navegación fija tipo terminal. Secciones como "paneles" de un dashboard financiero con bordes sutiles. Grid irregular con columnas de diferentes anchos.

**Elementos Firma:**
- Números que se animan al entrar en viewport (counter-up)
- Líneas de grid sutiles como papel milimetrado
- Gráficos con efecto de "scan line" retro

**Filosofía de Interacción:** Hover revela datos adicionales. Scroll suave entre secciones. Tooltips informativos en métricas.

**Animación:** Números con animación de conteo. Gráficos que se dibujan progresivamente. Fade-in escalonado de elementos. Efecto de "typing" en títulos clave.

**Sistema Tipográfico:** JetBrains Mono para datos/métricas, DM Serif Display para títulos, DM Sans para cuerpo de texto.

## Idea 2: "Obsidian Vault" — Brutalismo Financiero Elegante

**Movimiento de Diseño:** Neo-brutalismo refinado con influencias de diseño editorial suizo. Contraste extremo, tipografía masiva, composiciones asimétricas dramáticas.

**Principios Centrales:**
1. Contraste máximo — fondos negros puros con tipografía blanca masiva
2. Bordes duros y líneas rojas como separadores (inspirado en el PPTX original)
3. Imágenes de gráficos como piezas de arte dentro de marcos definidos
4. Jerarquía visual agresiva — lo importante es GRANDE

**Filosofía de Color:** Negro absoluto (#000000) como lienzo, blanco puro para texto principal, rojo carmesí (#dc2626) como acento único y poderoso. Sin grises intermedios — solo extremos.

**Paradigma de Layout:** Full-bleed sections alternando entre texto masivo a la izquierda y contenido a la derecha. Secciones con alturas de viewport completo. Composición editorial con columnas de texto estrechas junto a imágenes amplias.

**Elementos Firma:**
- Líneas rojas horizontales como separadores (del PPTX original)
- Tipografía de título que ocupa 40%+ del viewport
- Imágenes de charts con bordes definidos y sombras duras

**Filosofía de Interacción:** Scroll-triggered reveals dramáticos. Parallax sutil en imágenes. Hover que invierte colores.

**Animación:** Entradas con clip-path reveals. Texto que se desliza desde fuera del viewport. Transiciones de opacidad lentas y cinematográficas.

**Sistema Tipográfico:** Space Grotesk para títulos (bold, masivo), Instrument Serif para citas/destacados, IBM Plex Sans para cuerpo.

## Idea 3: "Quantum Edge" — Futurismo Financiero con Profundidad

**Movimiento de Diseño:** Diseño futurista inspirado en interfaces sci-fi y visualización de datos avanzada. Combina la seriedad institucional con una estética de tecnología de vanguardia.

**Principios Centrales:**
1. Capas de profundidad — glassmorphism sutil, sombras profundas, elementos flotantes
2. Gradientes oscuros sofisticados (no planos) — de navy a slate a negro
3. Acentos de cian/teal (#06b6d4) y esmeralda (#10b981) para datos
4. Geometría angular — clips diagonales, bordes cortados

**Filosofía de Color:** Gradiente base de #0f172a a #020617 (slate oscuro). Cian brillante para CTAs y datos positivos. Esmeralda para métricas de éxito. Blanco con opacidades variables para jerarquía textual.

**Paradigma de Layout:** Secciones full-width con contenido en containers asimétricos. Cards con glassmorphism flotando sobre fondos con gradientes. Layout de dos columnas desiguales (60/40) para secciones de contenido.

**Elementos Firma:**
- Cards con borde de gradiente (border-image)
- Partículas o puntos flotantes sutiles en el hero
- Métricas en "badges" con glow effect

**Filosofía de Interacción:** Cards que se elevan al hover con sombra expandida. Scroll progress indicator. Secciones que se revelan con parallax.

**Animación:** Entradas con spring physics (framer-motion). Números con morphing animation. Gradientes que se mueven sutilmente. Glow pulsante en elementos clave.

**Sistema Tipográfico:** Sora para títulos (geométrica, moderna), Inter para cuerpo (legibilidad), Fira Code para datos numéricos.
